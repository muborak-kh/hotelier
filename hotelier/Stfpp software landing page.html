Шахматка

Раздел системы, где будет отоброжатся информации о гостях в виде таблицы, где по горизонтали будет указаны даты и наличие номеров на эту дату, по вертикали номерной фонд гостиницы.

Интуитивно понятная форма шахматки, которая упрощает поиск оптимального варианта для гостя. На наглядном инструменте для отслеживания бронирований отображается актуальная информация о свободных номерах и тарифах. Программа позволяет проводить действия с бронированием, осуществлять переселение гостей, менять даты и время заезда/выезда. После занесения бронирования, система фиксирует данные и блокирует номер, делая невозможным его дальнейшую продажу. Статусы каждого номера за выбранный период времени для удобства пользователя выделяются разными цветами

Шахматка считается одним из главных функций системы, так как упрощает работы администраторов и сокращает время бронирование в несколько раз!
